# Learning-Flutter

## Português

Apps criados para aprender Flutter e testar diferentes componentes.

 - Lista de Tarefas: CRUD simples e elementos para criar uma lista
 - Message Chat: Criar interface de chat de mensagens simples
 - Soundboard: Botões que ativam sons 
 - Fetch Json: conversão de Json para dart e vice-versa

## English 

Apps created to learn Flutter and test different components

 - Todo List: Simple CRUD and elements to create a list
 - Message Chat: Create interface of a simple message chat
 - Soundboard: Buttons that activate sounds
 - Fetch Json: Json to dart conversion and vice-versa
